# Clothing Retail Multi-Store Manager

This is a single-page application for managing a multi-store clothing retail business. It is built with HTML, CSS, and vanilla JavaScript.

## Features

*   Dashboard with key metrics
*   Store management
*   Product catalog with variants
*   Inventory management
*   Purchase order management
*   Inter-store transfers
*   POS billing
*   Customer management
*   Sales and stock reports
*   Data import/export
*   Light and dark mode

## How to run

1.  Open the `index.html` file in a web browser.
2.  The application is ready to use.
