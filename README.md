# Multi-Store Manager - Universal Retail Management System

## Overview
This application has been updated to support **multiple product categories** beyond just clothing, making it suitable for any retail business.

## Supported Product Categories

### 1. **Clothing**
   - Men's wear (Tees, Jeans, Shirts)
   - Women's wear (Tops, Dresses)
   - Kids' wear (T-Shirts)

### 2. **Household Items** ✨ NEW
   - Kitchen (Utensils, Appliances)
   - Bathroom (Accessories)
   - Cleaning (Supplies)
   - Storage (Containers)
   - Decor (Items)

### 3. **Electronics** ✨ NEW
   - Mobile (Phones, Accessories)
   - Computers (Laptops, Accessories)
   - Audio (Headphones, Speakers)

### 4. **Groceries** ✨ NEW
   - Snacks, Beverages, Dairy
   - Frozen Foods, Packaged Foods
   - Fresh Produce

### 5. **Stationery** ✨ NEW
   - Writing (Pens, Pencils)
   - Paper (Notebooks, Sheets)
   - Art & Office Supplies

### 6. **Cosmetics & Beauty** ✨ NEW
   - Skincare, Makeup, Haircare
   - Fragrances, Personal Care
   - Beauty Tools

## Key Features

### Product Management
- **Product Type Selection**: Choose from 6 major product categories
- **Dynamic Categories**: Each product type has relevant subcategories
- **Flexible Variants**: 
  - For clothing: Size (XS-XXL) and Color
  - For household: Capacity/Weight and Material/Finish
  - For electronics: Storage/RAM and Color
  - For groceries: Pack Size and Flavor

### Smart Defaults
The system automatically adjusts input fields based on product type:
- **Clothing**: Traditional size (XS, S, M, L, XL, XXL) and color options
- **Household**: Capacity fields (500ml, 1L, 2L) and material options (Plastic, Steel, etc.)
- **Electronics**: Variant fields (64GB, 128GB) and color options
- **Groceries**: Pack size (100g, 250g, 500g, 1kg) and flavor variants

### No Variants Option
Products can be created without variants (e.g., single-variant items like groceries or household supplies)

## How to Use

1. **Create a New Product**:
   - Select Product Type (Clothing, Household, Electronics, etc.)
   - Choose appropriate Category from the dropdown
   - Fill in product details (Name, Brand, SKU, Price, etc.)
   - Add variants if needed (optional)
   - Click "Generate Variants" to preview
   - Save the product

2. **Filter Products**:
   - Use the Product Type filter in the product list
   - Search by name, SKU, brand, or category
   - View products grouped by type

3. **Load Demo Data**:
   - Click "Load Demo" button to see sample products
   - Includes examples from clothing, household, and electronics categories

## GST Compliance
All product types support:
- HSN/SAC codes
- GST percentage configuration
- Proper tax calculation on invoices

## Universal Features
- Multi-store inventory management
- Stock transfers between stores
